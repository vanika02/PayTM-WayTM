# Backend setup

## 1. Clone the repo
``` bash
git clone https://github.com/your-username/PayTM-WayTM.git
cd PayTM-WayTM/backend
```

## 2. Install Dependencies
``` bash
npm install
```

## 3. Set up environment variables
```bash 
cp .env.example .env
```

## 4. Start the server
```bash
npm run dev
```

## 5. Backend will run on
```bash 
http://localhost:5000
```
